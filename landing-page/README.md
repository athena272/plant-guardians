# PlantGuardians

Landing page para o PlantGuardians: proteção inteligente e sustentável para sua plantação.

## Tecnologias

- React + Vite
- SCSS (modular)
- React Icons

## Como rodar

```sh
npm install
npm run dev
```

## Sobre

Solução inovadora que utiliza IA e IoT para proteger plantações, reduzindo perdas e preservando a natureza.
